# Data_Analysis_Lab
A structured, execution-focused learning repository dedicated to mastering core data analytics tooling. This repository tracks practical milestones across data manipulation, relational database querying, and business intelligence visualization dashboards.

# 🎯 Learning Objectives
*The goal is to transition raw business data into actionable institutional insights by mastering the classic analytical stack:*

**Advanced Excel:** Data cleaning, logical functions, dynamic arrays, and structural modeling (VLOOKUP/XLOOKUP, Pivot Tables, Power Query).

**Relational Databases (SQL):** Constructing optimized queries, multi-table joins, aggregations, window functions, and subqueries to extract data from enterprise data warehouses.

**Business Intelligence (Power BI / Tableau):** Designing interactive, production-grade dashboards, establishing robust data models, and mapping key performance indicators (KPIs) for stakeholder review.

# 🗺️ Execution Phases
**Phase 1: Spreadsheets & Data Modeling (Excel)**
Focuses on handling initial data layers, automated cleaning routines, and quick diagnostic analysis.

Comprehensive data cleaning syntax and handling missing/duplicate variables.

Advanced analytical operations utilizing conditional aggregation (SUMIFS, COUNTIFS).

Dynamic dashboard prototyping using Pivot Charts and Slicers.

**Phase 2: Database Querying & Manipulation (SQL)**
Transitioning from flat files to enterprise database systems.

Writing structural data queries using standard SELECT, WHERE, GROUP BY, and HAVING clauses.

Joining complex relational tables (INNER, LEFT, RIGHT, OUTER).

Intermediate mechanics: Subqueries, Common Table Expressions (CTEs), and Window Functions (ROW_NUMBER, RANK, LEAD/LAG).

**Phase 3: Data Visualization & BI Architecture (Power BI & Tableau)**
Transforming structured query outputs into semantic layers and interactive visuals.

Connecting live databases and processing ETL (Extract, Transform, Load) operations.

Building relationships and star schemas within the data model.

Developing intuitive operational reports that track real-world business trends.
